This is a placeholder blog post.

To add a new post, place a Markdown file in `blog/posts/` with front matter:

```yaml
---
layout: post
title: "Your Title"
date: 2026-05-20
description: "One sentence summary."
---
```

After pushing to GitHub Pages, the blog index can list posts by date.
