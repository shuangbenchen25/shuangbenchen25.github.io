This is a placeholder blog post.

To add a new post, place a Markdown file in `src/content/blog/` with front matter:

```yaml
---
title: "Your Title"
date: 2026-05-20
description: "One sentence summary."
---
```

Astro now discovers these Markdown files during build and generates the blog list and post pages automatically.
